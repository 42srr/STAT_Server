## 42 SRR API Server

## 프로젝트 소개

- 42 본과정 활동에 관련된 정보를 가공하여 제공합니다.
- 개발기간 : 2024.04 ~


## 사용기술

- jdk 17
- Spring boot 3.2.5
- Spring MVC
- JPA / Hibernate
- MySQL, H2

## 기능 소개

### 1. Login 

OAuth2 기반 로그인 기능을 제공합니다.

### 2. User's Information

모든 사용자의 정보 또는 특정 사용자의 정보를 제공합니다.

### 3. Project Information

프로젝트에 대한 정보를 제공합니다. 

### 4. Ranking Information

랭킹 정보를 제공합니다.
